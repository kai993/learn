# DNS
## DNS
- Domain Name Systemの略
- IPアドレスは覚えられない
- DNSはドメイン名からIPアドレスに自動的に変換してくれる
- ホスト名とIPアドレスの関係を表すものをデータベースで管理している（分散データベース）
- IPv4, IPv6に対応してる
- ドメイン名とIPアドレスの関係性を明らかにすることを**名前解決**という
- ドメイン名からIPアドレスを求めることを**正引き**という
- IPアドレスからドメイン名を求めることを**逆引き**という

## ドメイン名
- ホストの名前や組織を識別するための階層構造(木構造)になってる
- yahoo.co.jpの場合
  - トップレベルドメイン(yahoo) : 固有のドメイン名
  - 分野別トップレベルドメイン(co) : 企業
  - 国コードトップレベルドメイン(jp) : 日本

## ネームサーバー
- ドメイン名を管理してるホストやソフトウェアのこと
- 階層ごとにネームサーバが置かれてる

## リゾルバ
- DNSに問い合わせを行うホストやソフトウェアのこと
