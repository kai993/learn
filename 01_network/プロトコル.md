## OSI参照モデル
- OSI参照モデルはあくまでもモデル
- 各層の大まかな役割を決めてる
- ガイドライン的なもの
- OSI参照モデルは7階層になってる

| 層                   | 機能                                                               | 説明                                                                                   |
|----------------------|--------------------------------------------------------------------|----------------------------------------------------------------------------------------|
| アプリケーション層   | 特定のアプリケーションに特化したプロトコル                         | アプリケーションの通信に関係する部分を定めてる。ファイル転送・メール・遠隔ログインなど |
| プレゼンテーション層 | 機器固有・ネットワーク共通のデータフォーマット                     | アプリケーションが扱う情報を通信に適したデータ形式に変換する。                         |
| セッション層         | 通信の管理をする。コネクションの確率・切断                         | コネクションの確率や切断、データの切れ目の設定など、データ転送に関する管理を行う       |
| トランスポート層     | 両端ノード間のデータ転送の管理をする。データ転送の信頼性を提供する | 宛先のアプリケーションにデータを確実に届ける。                                         |
| ネットワーク層       | アドレスの管理と経路を選択する                                     | 宛先にデータを届ける役割を持つ。                                                       |
| データリンク層       | 接続された機器間でデータフレームの識別と転送をする                 | 物理層で直接接続されたノード間での通信を可能にする                                     |
| 物理層               | 0と1を電圧の高低や光の点滅に変換する                               | ビット列を電圧の高低や光の点滅に変換する                                               |
